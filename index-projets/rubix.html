Jeu Unity 3D VR en Équipe

Projet final de cinquième session en TIM à Maisonneuve dans le cours d’Expérimentation en Jeu - volet Programmation réalisé en équipe de 4. Ce projet avait pour but d’expérimenter avec les technologies de réalité virtuelle dans Unity et avec les technologies URP sur Unity. Ce jeu a été conçu en fin de session d’automne 2023 (environ 1 mois).
Ce projet d’expérimentation prend la forme d’un jeu ayant un gameplay VR asymétrique. C’est-à-dire que le jeu se joue à deux. Le jeu doit être exécuté sur ordinateur, le premier joueur est sur la VR tandis que l’autre est à l’ordinateur. Ensemble, ils doivent résoudre des puzzles en communiquant les informations dont ils disposent respectivement. Le jeu étant un projet d’expérimentation, il n’est pas complet à proprement parler et n’a donc pas de menus.
Rôle: Programmeur (1/1)
